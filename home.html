Important FAQ

1. Q: What is DOM and how does it work?

Answer: DOM (Document Object Model) is a structured representation of a web page. It converts the HTML or XML document into a tree-like structure,
where each element is known as a "node". A page's content, style, and structure can be dynamically changed through the DOM using JavaScript.

2. Question: How to select an HTML element using DOM?

Answer:The DOM uses various methods to select HTML elements, such as:

  • getElementById
  • getElementsByClassName
  • querySelector
  • querySelectorAll
    • 3. Question: How to create a new element with DOM and add it to the page?

    Answer: New elements can be created and added using the DOM's createElement and appendChild methods

    4. Q: How to change the style of an element with DOM?

    Answer: Elements can be styled through CSS using the style property of the DOM.
